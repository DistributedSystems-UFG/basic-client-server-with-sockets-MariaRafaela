Calculadora

Para acrescentar complexidade e permitir que o cliente chame mais de uma funcionalidade diferente no servidor, foi implementada uma calculdadora simples (conforme sugerido no git) que realiza apenas 3 operações: soma, subtração e multiplicação. O cliente envia o comando contendo a operação e os números para o servidor ('operacao' 'n1' 'n2', sendo operação= "mul" ou "sum" ou "sub") e o servido responde com o resultado.

Exemplo: sum 1 2-> Resultado: 3.0

Além disso, o cliente pode digitar "fim" para finalizar a conexão.
